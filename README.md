# Heart Disease Prediction

## Project Overview
This project focuses on analyzing and exploring data related to heart disease to identify patterns and insights that may help in predicting heart disease.

## Tools & Technologies
- Python
- SAS Viya
- Data Analysis
- Data Cleaning & Preparation
- Exploratory Data Analysis (EDA)

## Project Stages
1. Data Understanding & Preparation  
2. Data Collection and Cleaning  
3. Exploratory Data Analysis (EDA)

## Deliverables
- Cleaned dataset
- Exploratory data analysis results
- Analytical report summarizing insights and findings
